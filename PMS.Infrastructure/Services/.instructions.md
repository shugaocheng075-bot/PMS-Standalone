# Service 实现编码约定

## 类定义
- 接口：`I{Module}Service` 放 `PMS.Application/Contracts/{Module}/`
- 实现：`InMemory{Module}Service : I{Module}Service` 放本目录
- DI 注册：`builder.Services.AddSingleton<I{Module}Service, InMemory{Module}Service>()` 在 `Program.cs`

## 数据持久化模式
```csharp
public class InMemoryXxxService : IXxxService
{
    private const string TableName = "Xxx";
    private const string LegacyJsonKey = "xxx_data";  // 旧 JSON 迁移键，新模块可省略
    private static readonly object SyncRoot = new();
    private static readonly List<XxxEntity> Records =
        SqliteTableStore.LoadAll<XxxEntity>(TableName, LegacyJsonKey);
    private static long _nextId = Records.Count > 0 ? Records.Max(x => x.Id) + 1 : 1;
}
```

## 硬性规则
1. **线程安全**：所有读写操作用 `lock (SyncRoot) { ... }` 保护
2. **持久化**：写操作后调用 `SqliteTableStore.Insert/Update/Delete<T>(TableName, entity)`
3. **查询链**：LINQ 多条件筛选 → `OrderByDescending` → `Skip((page-1)*size).Take(size)` → `Select(MapToDto)`
4. **数据范围**：通过 `query.AccessiblePersonnelNames` / `query.AccessibleHospitalNames` 做 HashSet 过滤
5. **返回 DTO**：永远不要直接返回 Entity，使用私有 `MapToDto` 方法转换
6. **Id 生成**：`Interlocked.Increment(ref _nextId)` 或 `lock` 内自增

## 查询过滤模板
```csharp
if (query.AccessiblePersonnelNames is { Count: > 0 })
{
    var nameSet = new HashSet<string>(query.AccessiblePersonnelNames, StringComparer.Ordinal);
    filtered = filtered.Where(x => nameSet.Contains(x.PersonnelName));
}
if (!string.IsNullOrWhiteSpace(query.Keyword))
    filtered = filtered.Where(x => x.Name.Contains(query.Keyword, StringComparison.OrdinalIgnoreCase));
```

## 禁止
- 不要引用 `PMS.API` 命名空间（Service 不能依赖 Controller 层）
- 不要直接访问 `HttpContext`（数据范围由 Controller 传入 Query DTO）
- 不要使用异步 I/O（SQLite 操作是同步的，方法签名为 async 仅为接口统一）

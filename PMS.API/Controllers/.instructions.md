# Controller 编码约定

## 类定义
- 使用 primary constructor 注入服务：`public class XxxController(IXxxService svc, IAccessControlService acs) : ControllerBase`
- 路由：`[ApiController]` + `[Route("api/{module}")]`
- 一个 Controller 文件只管一个业务模块

## 请求处理模式
```csharp
[HttpGet]
public async Task<IActionResult> Query([FromQuery] XxxQuery query, CancellationToken ct)
{
    var personnelId = HttpContext.GetCurrentPersonnelId();
    var dataScope = await accessControlService.GetDataScopeAsync(personnelId);
    // ... 业务查询 + 数据范围过滤
    return Ok(ApiResponse<T>.Success(result));
}
```

## 硬性规则
1. **所有方法 async/await + CancellationToken 参数**
2. **所有返回值用 `ApiResponse<T>.Success(data)` 包装**，失败用 `ApiResponse<T>.Fail(message)`
3. **数据隔离**：非 `all` 范围时必须调用 `HospitalScopeHelper.FilterByHospitalScope(dataScope, items, x => x.HospitalName)`
4. **获取当前用户**：`HttpContext.GetCurrentPersonnelId()`（来自 `HttpContextPermissionExtensions`）
5. **分页查询**：`[FromQuery] int page = 1, [FromQuery] int size = 20`
6. **文件导出**：返回 `File(bytes, "application/vnet.openxmlformats-officedocument.spreadsheetml.sheet", "filename.xlsx")`
7. **薄编排**：Controller 不写业务逻辑，委托给 Service

## 标杆文件
参考 `WorkHours/WorkHoursController.cs` 了解完整模式。

## 禁止
- 不要在 Controller 中直接操作 `SqliteTableStore` 或 `SqliteJsonStore`
- 不要引用 `PMS.Domain.Entities` 命名空间以外的 Domain 类型
- 不要跳过数据范围检查
